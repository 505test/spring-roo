gvNIX gvNIX is an open source tool for rapid application development (RAD) with which you can create Java web applications in minutes. It is a distribution of Spring Roo that provides the set of Spring Roo tools plus a suite of features that increase development productivity and improve the user experience by integrating frameworks like jQuery, Bootstrap 3, Leaflet, DataTables, Dandelion DataTables, among other.
