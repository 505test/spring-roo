Spring Roo is modular, you can install new features with the install addon command. A *Roo Addon Suite* is a great way to package and distribute a set of add-ons together, for example if you want to distribute Roo custom distributions. Roo Addon Suite is based on OSGi R5 Subsystems that provides a really convenient deployment model, without compromising the modularity of Roo. Add-ons are the most convenient way to reuse Spring Roo code, either commercially or free. Spring Roo page project serves as the store for the add-ons.
