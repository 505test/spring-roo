The recommended way to get started using spring-roo is download and unpackage zip on your system. 

<dependencies>
    <dependency>
        <groupId>org.springframework.roo</groupId>
        <artifactId>spring-roo</artifactId>
        <version>1.3.0.RELEASE</version>
    </dependency>
</dependencies>
After downloading, unpacking, and installing Roo, try out the following steps to create an application: ```sh mkdir hello cd hello roo roo> hint roo> project --topLevelPackage com.foo roo> jpa setup --provider HIBERNATE --database HYPERSONIC_IN_MEMORY roo> entity jpa --class ~.domain.Timer --testAutomatically roo> field string --fieldName message --notNull roo> hint web mvc roo> web mvc setup roo> web mvc all --package ~.web roo> selenium test --controller ~.web.TimerController roo> security setup roo> perform tests roo> quit ``` {% endcapture %} {% capture related_resources %} ### Additional Documentation * [Reference (PDF)](http://docs.spring.io/spring-roo/docs/current/reference/pdf/spring-roo-docs.pdf) ### Videos * [Roo in Five Minutes](http://s3.springsource.com/MRKT/roo/2010-01-Five_Minutes_Roo.mov) ### Books * [Spring Roo in Action](http://www.manning.com/rimple/) * [Getting Started with Roo](http://shop.oreilly.com/product/0636920020981.do) * [Spring Roo 1.1 Cookbook](http://oreilly.com/catalog/0636920020981) ### Project Lead * [DISID Corporation](http://www.disid.com/en/) {% endcapture %} {% include project_page.html %}
